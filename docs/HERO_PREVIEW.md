# Hero Preview Guide

The README is prepared for a hero image at:

```text
docs/images/hero-preview.png
```

## Recommended image

- Size: approximately 1600 × 800 or 1600 × 900 pixels
- Format: PNG
- Show a useful static GLB model in 3ds Max 2016 and the exported result in an independent viewer
- Prefer a clean side-by-side composition
- Remove browser tabs, bookmarks, personal folders, usernames, and unrelated desktop elements
- Avoid tiny text that becomes unreadable on GitHub
- Use a model you created or an asset whose license allows public promotional screenshots

## Suggested composition

Left side:

- 3ds Max 2016 viewport
- Imported model with materials visible
- A small hint of the Material Editor or import summary, without clutter

Right side:

- The exported GLB in an independent viewer
- Same camera angle when possible

Optional title area:

```text
MaxGLB2016
Static GLB import and export for 3ds Max 2016
```

## Enabling the image

After adding the final PNG, uncomment the prepared HTML image block near the top of `README.md`.
